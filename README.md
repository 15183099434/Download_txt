## 非要说有什么特点的话...
##### 下载速度贼慢，不知道是我网不稳定的问题还是目标网站服务器的问题
##### 下载的小说适用于ios的qq浏览器里的小说书架中的小说格式（其他设备我没有，没试过，电脑的文本文档没有分好行，很难受{其实可以用ide看..ide里是分好了行的}）
##### 少数页面转换格式时会出错（因为有一些转不了gbk的特殊符号）
##### 简陋，如果真有人用的话，大概可以养成您用浏览器下小说的好习惯
